# 反思维度：周期复盘 (Periodic Review)

> "回顾是智慧的母亲。" —— 乔治·洛里默

---

## 周期复盘的本质

周期复盘是从更大的时间尺度审视过去，是将碎片洞察整合为系统智慧的质量控制机制。不同周期的复盘有不同的目的和深度。

**复盘周期的层次**：

```
┌─────────────────────────────────────────────────────┐
│              复盘周期体系                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│  日复盘 ←→ 周复盘 ←→ 月复盘 ←→ 季复盘 ←→ 年复盘  │
│                                                     │
│  │        │        │        │        │             │
│  细节    趋势    模式    战略    方向              │
│  日      周      月      季      年                │
│                                                     │
└─────────────────────────────────────────────────────┘

日复盘：细节校准
周复盘：趋势识别
月复盘：模式分析
季复盘：战略调整
年复盘：方向重置
```

## 周复盘 (Weekly Review)

### 周复盘的目的

```
1. 超越每日视角，看到更大pattern
2. 从数据中提取趋势与规律
3. 校准方向与优先级
4. 清空累积的"心理缓存"
5. 计划下周的核心行动
```

### 周复盘执行指南

```
时间：每周日 19:00-20:30

流程：
1. 收集数据（10分钟）
   - 回顾本周日记
   - 查看日历事件
   - 回顾待办完成情况

2. 分析模式（20分钟）
   - 什么做得好？为什么？
   - 什么做得不好？为什么？
   - 有什么新的发现？

3. 提炼洞察（10分钟）
   - 本周学到了什么？
   - 这些洞察与什么相关？

4. 计划下周（20分钟）
   - 确定TOP3目标
   - 安排时间块
   - 设定习惯承诺

5. 系统维护（15分钟）
   - 清空收集箱
   - 更新项目状态
   - 处理积压

6. 感恩时刻（10分钟）
   - 记录本周值得感恩的事
   - 庆祝完成
```
