## 快速参考

### 五维度核心原则

**方向 (Direction)**
> 方向优先于行动，没有清晰方向的行动是盲目的

**行动 (Action)**
> 微小一致的执行，累积巨大的成果

**认知 (Cognition)**
> 知识需要转化才能产生价值

**反思 (Reflection)**
> 不经审视的经验只是经历

**资源 (Resources)**
> 失去健康，其他维度无从谈起
