# 五维度框架决策框架

> 系统化的决策方法论

---

## 决策的本质

决策是在资源有限的情况下，选择最优行动的过程。好的决策能够推动目标实现，坏的决策则会导致资源浪费和机会成本。

**决策的核心要素**：
- 选项：有什么选择？
- 信息：知道什么？
- 价值观：什么更重要？
- 结果：每个选项的后果是什么？
- 风险：不确定性如何？

---

## 决策层级

### 第一层：日常决策

**频率**：每天多次
**影响**：小到中等
**时间投入**：秒到分钟级

```
日常决策示例：
- 现在做什么？
- 这封邮件回还是不回？
- 这个会议要不要参加？
- 午饭吃什么？
```

**方法**：直觉 + 简单规则

```
决策规则：
- 2分钟能做完的立即做
- 不能连接目标的就不做
- 不确定时选择健康而非工作
```

---

### 第二层：战术决策

**频率**：每周几次
**影响**：中等
**时间投入**：分钟到小时级

```
战术决策示例：
- 这个任务放这周还是下周？
- 要不要接受新的项目？
- 如何分配团队资源？
- 这个工具要不要用？
```

**方法**：成本收益分析 + 优先级矩阵

---

### 第三层：战略决策

**频率**：每月/每季
**影响**：大
**时间投入**：小时到天级

```
战略决策示例：
- 这个季度目标是否调整？
- 要不要转型方向？
- 重大投资决策
- 人员/组织调整
```

**方法**：多维度评估 + 长期视角

## 日常决策框架

### 决策矩阵：做什么 vs 不做什么

```
问自己：
1. 这件事必须我做吗？
2. 这件事对目标有贡献吗？
3. 这件事紧急吗？
4. 我现在有精力做吗？
```

```
决策结果：
├── 必须做 + 对目标有贡献 + 紧急 → 立即做
├── 必须做 + 对目标有贡献 + 不紧急 → 安排时间做
├── 必须做 + 对目标无贡献 → 委托或简化
├── 不必须做 → 不做或推迟
```

---

### 优先级决策：先做什么

**艾森豪威尔矩阵**：

```
                    紧急
              低　　　　　　高
          ┌───────┬───────┐
      高　│ 执行  │ 优先  │
重要性    ├───────┼───────┤
      低　│ 忽略  │ 委托  │
          └───────┴───────┘
```

**应用**：
```
优先（紧急+重要）：立刻做
- 截止日期临近
- 危机处理
- 重要会议准备

执行（不紧急+重要）：计划做
- 长期目标相关
- 能力建设
- 预防性工作

委托（紧急+不重要）：交给别人
- 别人也能做
- 不需要你的独特能力

忽略（不紧急+不重要）：不做
- 纯消遣
- 无意义社交
```

---

### 情境决策：怎么做

**基于当前状态选择**：

```
能量状态：
- 高能量 → 做复杂/创造性任务
- 中能量 → 做常规任务
- 低能量 → 做简单/机械任务

时间多少：
- 充足 → 深入做
- 有限 → 快速处理
- 紧迫 → 先完成核心

情境：
- @电脑 → 处理需要电脑的任务
- @外出 → 处理在外可以做的事
- @电话 → 处理需要通话的事
```

### 人际决策：与谁互动

```
决策问题：要不要参加某个社交活动？

评估维度：
1. 能量收益：会让我更有能量还是消耗能量？
2. 关系价值：对我重要的人会参加吗？
3. 时间成本：这个时间用在别处是否更有价值？
4. 成长机会：能学到新东西或拓展视野吗？

决策：
- 能量收益高 + 关系价值高 → 参加
- 时间成本高 + 成长机会低 → 拒绝
- 不确定 → 询问细节后再决定
```

## 战术决策框架

### 项目取舍

**决策问题**：要不要接这个新项目？

```
评估框架：

1. 战略对齐
   - 与我的方向/目标相关吗？
   - 权重：30%

2. 资源需求
   - 需要多少时间/精力/金钱？
   - 我有足够资源吗？
   - 权重：25%

3. 机会成本
   - 做这个会失去什么？
   - 同样的时间能用在更重要的事上吗？
   - 权重：20%

4. 风险评估
   - 可能出问题吗？
   - 我能承受最坏结果吗？
   - 权重：15%

5. 退出机制
   - 如果不顺利能退出吗？
   - 有没有承诺无法撤回？
   - 权重：10%
```

**评分**：
```
每项1-5分
加权总分 = Σ(分数 × 权重)

结果：
- ≥4分：强烈推荐
- 3-4分：可以考虑
- <3分：谨慎或拒绝
```

---

### 工具/订阅决策

**决策问题**：要不要订阅/购买这个工具？

```
评估框架：

1. 必要性
   - 现在真正需要吗？
   - 有替代方案吗？
   - 权重：30%

2. 使用频率
   - 会经常用吗？（>每周2次）
   - 能坚持使用吗？
   - 权重：25%

3. 成本收益
   - 月成本是多少？
   - 带来的价值值得吗？
   - 权重：25%

4. 转换成本
   - 迁移容易吗？
   - 数据能导出吗？
   - 权重：10%

5. 扩展性
   - 以后会更需要吗？
   - 能满足未来需求吗？
   - 权重：10%
```

---

### 人脉决策

**决策问题**：要不要主动联系某人？

```
评估框架：

1. 关系深度
   - 多久没联系了？
   - 关系重要程度？

2. 连接理由
   - 有具体的事要聊吗？
   - 还是只是维护关系？

3. 能量方向
   - 这个联系会让我更有能量吗？
   - 对方也会受益吗？

4. 时间投入
   - 需要多少时间？
   - 这个时间用在别处更好吗？

决策：
- 重要关系 + 很久没联系 + 有理由 → 主动联系
- 一般关系 + 没有具体事由 → 等有理由再说
- 消耗能量的关系 → 谨慎或避免
```

## 战略决策框架

### 目标调整

**决策问题**：要不要调整季度目标？

```
评估标准：

1. 环境变化
   - 有什么重大变化发生了吗？
   - 原目标的前提是否仍然成立？

2. 进展评估
   - 目标完成率如何？
   - 差距是因为什么？

3. 调整成本
   - 调整目标需要什么代价？
   - 不调整继续做的代价是什么？

4. 方向影响
   - 调整会影响长期愿景吗？
   - 会让别人失望吗？

5. 时机
   - 现在调整是否太早/太晚？
   - 下次审视是什么时候？

决策原则：
- 调整比坚持错误目标好
- 但不要频繁调整
- 每次调整都要有明确的理由
```

---

### 资源配置

**决策问题**：如何分配有限资源（时间/金钱/精力）？

```
配置原则：

1. 方向优先
   - 资源优先投入最重要的事
   - 方向决定了什么是"重要"

2. 边际收益
   - 当资源在一个领域已经足够时，转移到其他领域
   - 不要在低价值地方过度投入

3. 缓冲预留
   - 预留20%时间作为缓冲
   - 不要排得太满

4. 定期重评
   - 每季度审视资源配置
   - 根据进展调整

配置公式：
```
可用资源 = 总资源 × (1 - 缓冲比例)

优先级分配：
1. 必须支出（生存/健康）：X%
2. 战略投入（方向相关）：Y%
3. 弹性使用（可选）：Z%

基于阶段性目标调整比例
```
```

### 重大取舍

**决策问题**：面临两难选择怎么办？

```
框架：画线法

1. 明确底线
   - 什么是你绝对不能妥协的？
   - 什么是你无论如何都要保住的？

2. 识别核心诉求
   - 两个选项背后各代表什么？
   - 哪个更接近你的核心价值？

3. 时间视角
   - 1年后怎么看这个选择？
   - 10年后呢？

4. 最小化遗憾
   - 如果选错了，哪种遗憾更难承受？
   - 哪个选择让你更接近理想？

决策：
- 有明确底线 → 底线优先
- 没有底线 → 选更能接近核心价值的
- 两边都有遗憾 → 接受"完美不存在"
```

### 决策质量评估

```
评估问题：

1. 信息充分吗？
   - 决策时有多少信息是确定的？
   - 有没有遗漏重要信息？

2. 选项完整吗？
   - 有没有第三个选项？
   - 有没有"不作为"的选项？

3. 分析理性吗？
   - 情绪是否影响了判断？
   - 偏见是否影响了分析？

4. 行动执行了吗？
   - 决策后是否立即行动？
   - 是否有拖延？

5. 结果复盘了吗？
   - 实际结果和预期对比如何？
   - 学到了什么？
```

## 决策陷阱与应对

### 陷阱一：分析瘫痪

```
症状：想太多，做不了决定

原因：
- 追求完美信息
- 害怕做出错误选择
- 过度分析

应对：
- 设定决策时限
- "足够好"而非"完美"
- 接受"犯错也是学习"
```

### 陷阱二：确认偏见

```
症状：只听到想听的

原因：
- 先有结论再找理由
- 忽视反对证据

应对：
- 主动寻找反对证据
- 问"如果我错了呢？"
- 让别人挑战你的判断
```

### 陷阱三：近期偏见

```
症状：过度重视近期，忽视长期

原因：
- 近期的痛苦/快乐更强烈
- 未来太远不确定

应对：
- 问"1年后怎么看？"
- 问"10年后呢？"
- 用成本收益量化长期价值
```

### 陷阱四：社会认同

```
症状：跟随别人的选择

原因：
- 不想与众不同
- 认为多数人是对的

应对：
- 独立思考：这是适合我的吗？
- 区分"建议"和"事实"
- 知道自己为什么选择
```
